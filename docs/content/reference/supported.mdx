# Supported Dialects & Libraries

## Dialects

Imported from `rosetta-date/dialects`.

| Dialect  | Grammar | Example | Literals |
| -------- | ------- | ------- | -------- |
| `moment` | [Moment.js](https://momentjs.com/docs/#/displaying/format/) token grammar | `DD/MM/YYYY` | `[...]` |
| `ldml`   | [Unicode UTS #35 / LDML](https://www.unicode.org/reports/tr35/tr35-dates.html#Date_Field_Symbol_Table) date field symbols | `dd/MM/yyyy` | `'...'` |

## Libraries

Imported from `rosetta-date/libraries`.

| Library    | Tool | Speaks | Coverage |
| ---------- | ---- | ------ | -------- |
| `momentjs` | [Moment.js](https://momentjs.com/docs/#/displaying/format/) | `moment` | the full grammar |
| `dayjs`    | [Day.js](https://day.js.org/docs/en/display/format) | `moment` | a core subset + the common plugins (AdvancedFormat, LocalizedFormat) |
| `dateFns`  | [date-fns](https://date-fns.org/docs/format) | `ldml` | the full grammar **+ its own extensions** (`P…`, `t`/`T`, `R`/`I`/`i`); some tokens gated behind date-fns options |

For the per-token grammar tables, see [Token Mapping](/reference/token-mapping). For tool-specific
behaviour, see [Library Notes](/reference/library-notes).
